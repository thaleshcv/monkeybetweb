# README

## Ruby

Instalar RVM [https://rvm.io/rvm/install]:

linux:

`curl -sSL https://get.rvm.io | bash`

Deslogar e logar novamente.

Instalar Ruby:

`rvm install ruby-3.1.3`

`rvm --default use 3.1.3`

`gem install bundle`

Apenas produção: `bundle config set --local deployment 'true'`

`bundle install`
